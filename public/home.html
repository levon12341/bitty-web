Bitty

Bitty is a free, open-source project that helps you earn money with Bitcoin purchasing! Moreover, we are predicting an approximate BTC price, so now, using our services you can make a decision whether you should buy Bitcoin or not!

Yes, most of people would say that predicting cryptocurrency value is impossible. But we have another point of view. We believe that using advanced forecasting models cryptocurrency can be predicted with great accuracy. Check our forecasting services to confirm this!

How does it work?

We are using advanced machine learning techniques(esp. Gradient Boosting) to predict bitcoin prices for you. All code is stored on our github.

Can I use your data?

Yes, you definitely can! We've build our REST API for that.

  • Get current btc value:
    $ curl api.bittie.ru/forecast/currencyValue
  • Get current BTC price prediction:
    $ curl api.bittie.ru/forecast/prediction
  • Get list of all BTC prices from given time:
    $ curl api.bittie.ru/forecast/listValuesFromTime?fromTime=FROM_TIME #default is 0
  • Get list of all predictions from given time:
    $ curl api.bittie.ru/forecast/listPredictionsFromTime?fromTime=FROM_TIME #default is 0
